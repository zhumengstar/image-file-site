## 产品概述

为图床网站侧边栏图片列表添加拖拽排序功能，用户可以通过拖拽改变图片的排列顺序。

## 核心功能

- **列表视图拖拽排序**：在列表视图模式下，支持拖拽图片项改变顺序
- **瀑布流视图拖拽排序**：在网格视图模式下，同样支持拖拽排序
- **视觉反馈**：拖拽过程中显示拖拽状态和放置位置指示
- **持久化提示**：排序后显示成功提示（由于后端未支持排序持久化，当前仅前端排序）
- **兼容现有功能**：保留原有的点击查看大图、复制链接等功能

## 技术栈

- **前端**：React 18 (CDN) + Babel (CDN)
- **样式**：CSS3
- **拖拽实现**：HTML5 Drag and Drop API

## 实现方案

使用 HTML5 原生拖拽 API 实现排序功能：

1. **拖拽状态管理**

- 添加 `draggedIndex` state 记录当前拖拽的图片索引
- 添加 `dragOverIndex` state 记录拖拽经过的目标位置

2. **事件处理**

- `dragStart`：记录拖拽开始的图片索引，设置拖拽数据
- `dragOver`：阻止默认行为，计算并更新目标位置索引
- `drop`：执行图片顺序交换，更新 images state
- `dragEnd`：清理拖拽状态

3. **排序算法**

- 使用数组 `splice` 方法移除拖拽项并插入新位置
- 保持其他图片的相对顺序不变

4. **视觉反馈**

- 拖拽项半透明化（opacity: 0.5）
- 目标位置显示插入指示线（边框高亮）
- 添加过渡动画使排序更流畅

## 实现注意事项

- **性能优化**：使用 `e.preventDefault()` 避免拖拽时的页面滚动
- **事件冒泡**：确保复制按钮的点击事件不会触发拖拽
- **视图兼容**：列表和网格视图使用相同的拖拽逻辑，但样式表现不同
- **移动端适配**：考虑添加触摸事件支持（后续可扩展）
